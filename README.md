# PHP Library Fastest

Fastest router

## Background ##

## Installation ##

Add the following to your composer file:

```json
   "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sinevia/php-library-fastest.git"
        }
    ],
    "require": {
        "sinevia/php-library-fastest": "dev-master"
    },
```

## Usage ##


```php
\Sinevia::fastest();
```
